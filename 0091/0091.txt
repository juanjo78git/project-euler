0091

Los puntos P(x1, y1) Y Q(x2, y2) están representados en coordenadas enteras
y se unen con el origen, O(0,0), para formar ΔOPQ.

Existen exactamente catorce triángulos con un ángulo recto que se forman cuando
las coordenadas se encuentran entre 0 y 2 inclusive; es decir,
0 ≤ x1, y1, x2, y2 ≤ 2.

Dado 0 ≤ x1, y1, x2, y2 ≤ 50, ¿cuántos triángulos rectángulos se pueden formar?
